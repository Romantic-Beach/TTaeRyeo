# TTaeRyeo

Arena Battle Game for C++ &amp; Arduino

### 실행 전 설치해야 될 것

\*\*\* 주의 애플 실리콘에서만 됨

1. CMake

```
brew install cmake
```

2. Boost

```
brew install boost
```

### 프로젝트 실행 방법

1. 경로 옮기기

```
cd ./client
```

2. cmake 명영어 실행

```
cmake .
```

3. make 파일 실행

```
make .
```

4. 만들어진 파일 실행

```
./GAB
```
